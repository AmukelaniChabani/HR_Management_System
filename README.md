# HR_Management_System
My first project an HR_Management_System which will help a company manage employees  you can ADD,Remove,Update and view Employees


How the system works 
1. The System is connected to a database and communicates the needs of the user with 
the data base than the database stores the  information, when the data is needed by 
the user the HRMS(Human Resource Management System ) than retrieve’s the data 
from the database 
2. When the user starts a process like creating an Account for example they fill a form with 
their information that information is than sent to the to the  HRMS(Human Resource 
Management System ) for validation than if the data sent is ok, than its sent to the data 
base to be Stored
